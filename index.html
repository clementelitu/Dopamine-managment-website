Install My Chrome Extension

This is a private test build for friends & family.

1) Download

Download the extension ZIP  ·  SHA-256 checksum

2) Install (Chrome)

  1. Unzip the downloaded file.
  2. Open chrome://extensions
  3. Turn on Developer mode (top-right).
  4. Click Load unpacked
  5. Select the unzipped folder (the one containing manifest.json).

Troubleshooting

  • If you don’t see “Load unpacked”, Developer mode isn’t enabled.
  • If it errors, double-check the selected folder contains manifest.json.
