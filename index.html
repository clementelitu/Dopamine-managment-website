Social Media Manager (Private Test)

Install the Chrome extension manually (friends & family build).

Manifest V3 Manual install

Download

Download the ZIP, unzip it, then load it via chrome://extensions.

⬇️ Download SocialMediaManager.zip

If the download ever 404s, it usually means the filename/path doesn’t match exactly (case-sensitive).

Install steps

  1. Unzip SocialMediaManager.zip to a folder.
  2. Open chrome://extensions
  3. Enable Developer mode (top-right).
  4. Click Load unpacked
  5. Select the unzipped folder that contains manifest.json.
Tip: If Chrome shows an error, make sure you selected the folder containing manifest.json (not the parent folder).

Need to uninstall? Go to chrome://extensions and click Remove.
